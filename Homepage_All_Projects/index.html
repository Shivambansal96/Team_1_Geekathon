Chrome Extension

Browser extensions are able to modify Google Chrome. They are supported by the browser's desktop edition. These extensions are written using web technologies like HTML, JavaScript, and CSS. They are distributed through Chrome Web Store, initially known as the Google Chrome Extensions Gallery. Some extensions focus on providing accessibility features.

Resume Template

A resume template is a pre-formatted document that serves as a blueprint for creating a professional resume. It typically includes sections for personal information, education, work experience, skills, and other relevant information. The layout and design of a resume template can vary depending on the industry and the preferences of the user.

Invoice Generator

An invoice generator, also known as an invoice maker, is an online tool that allows small businesses, freelancers or contractors to generate invoices quickly and efficiently. The tool provides standard formatting, layout, and content, making the overall billing process more efficient and quick1. The invoices generated can be sent and paid online or downloaded as a PDF

Minesweeper Game

Minesweeper is a logic puzzle video game genre generally played on personal computers. The game features a grid of clickable tiles, with hidden "mines" (depicted as naval mines in the original game) scattered throughout the board.

Code Editor
